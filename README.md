# MathService. Sequence generator

Sequence generator
 
### Using the service

Start service:

	mathCosSequence.jar
	
or

	java -jar mathCosSequence.jar
	
Open the link:

	http://localhost:8080/math/cos/cosSequenceGenerator?n=3

### Configuration

the same location as **mathCosSequence.jar**

* application.properties

    	server.port=8080
